# <a name="header-details-for-project-quotes"></a>Сведения о заголовке для предложений с расценками по проекту

_**Относится к:** развертывание Lite — от сделки до счетов-проформ_

В этой статье объясняется информация, которая относится к предложению с расценками по проекту. Это включает в себя настройки, которые влияют на все строки предложения с расценками, и информацию о предложении с расценками, которая суммируется по всем позициям строк для управления ключевыми показателями эффективности предложения с расценками по проекту.

В следующей таблице перечислены поля сводной информации в предложении по проекту, которые уникальны для Dynamics 365 Project Operations или содержат важные изменения в поведении по сравнению с предложениями с расценками в Dynamics 365 Sales.

| **Поле** | **Расположение** | **Описание** | **Воздействие на последующие элементы** |
| --- | --- | --- | --- |
| Тип | Вкладка "Сводка" (скрытая) | Это поле набора параметров имеет следующие параметры:</br>- На основе работы (доступно только при установленном Project Operations)</br>- На основе номенклатуры (доступно только при установленных Project Operations и Sales)</br>- На основе сервисного обслуживания (доступно, если установлен Dynamics 365 Field Service) | Когда вы используете приложение Project Operations, для этого поля автоматически устанавливается значение **На основе работ**. Это классифицирует предложение с расценками как предложение с расценками на основе проекта. Предложение с расценками должно быть основано на проекте, чтобы включить все расширения и функции, специфичные для проекта. |
| Потенциальный клиент | Вкладка сводных данных | Ссылка на запись компании или учетную запись клиента. Когда предложение с расценками создается из возможной сделки, это поле копируется из соответствующего поля в возможной сделке. | Валюта в предложении с расценками по проекту устанавливается по умолчанию в зависимости от валюты клиента. Это можно, однако, изменить перед сохранением предложения с расценками. |
| Менеджер по работе с клиентами | Вкладка сводных данных | Имя менеджера по обслуживанию клиентов для этой сделки. Когда предложение с расценками создается из возможной сделки, это поле копируется из соответствующего поля в возможной сделке. | Менеджер по работе с клиентами отвечает за управление отношениями с клиентом до завершения этого проекта. На основе записи резервируемого ресурса, связанного с менеджером по работе с клиентами, контрактная единица задается по умолчанию в предложении с расценками по проекту. |
| Единица по контракту | Вкладка сводных данных | Подразделение, отвечающее за реализацию проекта или проектов, связанных с этим предложением с расценками. Когда предложение с расценками создается из возможной сделки, это поле копируется из соответствующего поля в возможной сделке. | Подрядное подразделение — это подразделение компании, которое будет выполнять проекты после закрытия сделки. У каждой контрактной единицы есть валюта, и эта валюта используется для отчета о предполагаемых и фактических затратах, понесенных в ходе реализации проекта. |
| Прайс-лист продуктов | Вкладка сводных данных | Это прайс-лист, который используется для определения цен по умолчанию в строках предложений с расценками на основе продуктов. В списке параметров для этого поля отображается список прайс-листов, в которых валюта прайс-листа совпадает с валютой в предложении с расценками. Когда предложение с расценками создается из возможной сделки, это поле копируется из соответствующего поля в возможной сделке. Это поле возможной сделки устанавливается по умолчанию из записи учетной записи, но может быть изменено. | Когда предложение с расценками выиграно, значение этого поля копируется в контракт по проекту, который создается. |
| Валюта | Вкладка сводных данных | Это указывает на валюту, которая будет использоваться для отчета о сумме сделки. Это также валюта, в которой клиенту будет выставляться счет, если сделка будет реализована. Когда предложение с расценками создается из возможной сделки, это поле копируется из соответствующего поля в возможной сделке. Это поле возможной сделки устанавливается по умолчанию из записи учетной записи, но может быть изменено пользователем. | После сохранения предложения с расценками это поле больше не доступно для редактирования. Это используется для задания по умолчанию прайс-листов на продукты и проекты в предложении с расценками. Валюта в предложении с расценками используется в соответствии с валютой прайс-листа. |
| Максимальный лимит | Вкладка сводных данных | Это указывает на согласованный верхний предел окончательной стоимости, с которым клиент соглашается для этой сделки. | Это ограничение оценивается во время выполнения и применяется ко всем позициям строк и проектам, связанным с этой сделкой. |
| Запрошенная дата доставки | Вкладка сводных данных | Когда предложение с расценками создается из возможной сделки, это поле копируется из соответствующего поля в возможной сделке. | Эта дата используется в качестве конечной даты для создания графиков счетов. |

Ниже приведены вкладки и ключевые показатели эффективности, доступные в предложении с расценками по проекту, которые являются уникальными для Project Operations или имеют некоторые важные изменения в поведении по сравнению с предложениями с расценками из Sales:

| **Поле** | **Расположение** | **Описание** |
| --- | --- | --- |
| Анализ прибыльности | Вкладка в предложении с расценками | На вкладке отображаются указанные ниже метрики:</br>- Общая стоимость, подлежащая оплате</br></br>- Общая стоимость, не подлежащая оплате</br>- Общий доход</br>- Общий доход (базовый)</br>- Валовая прибыль</br>- Скорректированная валовая прибыль|
| Сравнение с ожиданиями клиента | Вкладка в предложении с расценками | На этой вкладке отображаются указанные ниже метрики:</br>- Ожидаемое завершение</br>- Запрошенное завершение</br>- Бюджет клиента</br>- Значение по предложению с расценками |
| Анализ предложения с расценками | Вкладка в предложении с расценками | На этой вкладке приведена сводка следующих основных ключевых показателей эффективности для предложения с расценками по проекту</br>- Сравнение с ожиданиями клиента по бюджету и графику</br>- Валовая прибыль</br>- Скорректированная валовая прибыль |
