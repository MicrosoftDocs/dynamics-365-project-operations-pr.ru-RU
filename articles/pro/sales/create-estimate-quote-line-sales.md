# <a name="estimating-a-project-based-quote-line"></a>Оценка строки предложения с расценками по проекту

_**Относится к:** Project Operations для сценариев на основе ресурсов/нескладируемых запасов, упрощенное развертывание — от сделки до выставления счетов-фактур_

В строке предложения с расценками на основе проекта есть детали, которые помогают оценить стоимость и потенциальную выручку от работы, связанной с доставкой строки предложения с расценками.

Чтобы оценить строку предложения с расценками на основе проекта, в строке предложения с расценками на основе проекта выберите вкладку **Сведения строки предложения с расценками**. Есть два способа создать оценку для строки предложения с расценками на основе проекта:

- Создайте оценку вручную прямо в строке предложения с расценками, используя сведения строки предложения с расценками 
- Создайте проект и план проекта, затем свяжите проект и задачи проекта со строкой предложения с расценками. Будет включен процесс импорта оценок из плана проекта в строку предложения с расценками на основе предоставленной вами информации.

## <a name="create-estimates-directly-on-a-project-based-quote-line"></a>Создание оценок непосредственно в строке предложения с расценками на основе проекта

Чтобы создать оценку для строки предложения с расценками на основе проекта, выберите вкладку **Сведения строки предложения с расценками**. Позиция строки, которую вы создаете на этой вкладке, будет суммировать указанное значение для этой строки предложения с расценками. 

Чтобы создать сведения о строке предложения с расценками, выберите **+ Создать сведения строки предложения с расценками** на вложенной сетке **Сведения строки предложения с расценками**. Откроется ползунок быстрого создания. Следующие поля в форме **Строка предложения с расценками**:

## <a name="update-prices-on-quote-line-details"></a>Обновление цен в сведениях строки предложения с расценками

Если вы изменили цены в прайс-листе проекта, который прилагается к предложению с расценками, или в прайс-листе себестоимости контрактной единицы, вы можете выбрать **Пересчитать** на странице **Предложение с расценками**, чтобы обновить цены в сведениях отдельной строки предложения с расценками, чтобы отразить это изменение. Когда вы выбираете **Пересчитать**, появляется предупреждение, информирующее вас о том, что цены в сведениях строки предложения с расценками для всех строк предложения с расценками в этом предложении с расценками будут сброшены. Выберите **Да**, чтобы обновить цены как для продаж, так и для сведений о затратах строки предложения с расценками.

## <a name="access-quote-line-details-for-cost"></a>Доступ к сведениях строки предложения с расценками для стоимости

На вкладке **Сведения строки предложения с расценками** выберите строку в сетке, чтобы включить некоторые действия на панели инструментов вложенной сетки. Первое действие на панели инструментов вложенной сетки, когда выбраны сведения строки предложения с расценками, — это **Открыть детализацию затрат**. Выберите **Открыть сведения о стоимости**, чтобы увидеть соответствующую ставку затрат и сумму для этой строки предложения с расценками.

> [!NOTE]
> Изменение значений единиц ресурса, количества, дат, роли или категории в сведениях строки предложения с расценками для стоимости приведет к изменению соответствующих значений в сведениях строки предложения с расценками для продаж.
## <a name="currency-on-quote-line-details-for-cost-and-sales"></a>Валюта в строке предложения с расценками для затрат и продаж

Валюта в сведениях строки предложения с расценками для продаж по умолчанию берется из прайс-листа проекта, который действует на дату начала сведений строки предложения с расценками.

Валюта в сведениях строки предложения с расценками для стоимости по умолчанию берется из прайс-листа контрактной единицы проекта, который действует на дату начала сведений строки предложения с расценками для стоимости.

При расчетах рентабельности сумма в сведениях строки предложения с расценками для стоимости и продаж преобразуется в базовую валюту среды, чтобы сообщить об общей предполагаемой марже по предложению с расценками.

Это может привести к ошибкам округления валют и изменению маржи из-за отсутствия действующих на данную дату обменных курсов. Используйте эти расчеты для предложений с расценками по проекту только как приблизительные, а не как фактические обязательную или иную отчетность, которая требует более высокой точности округления и учета даты действия для обменных курсов.
