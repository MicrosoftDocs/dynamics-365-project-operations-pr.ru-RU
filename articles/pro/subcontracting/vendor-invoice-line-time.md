# <a name="vendor-invoice-lines-for-time"></a>Строки счета поставщика для времени

[!include [banner](../../includes/dataverse-preview.md)]

_**Относится к:** развертывание Lite — от сделки до счетов-проформ_

Счет поставщика в Microsoft Dynamics 365 Project Operations может содержать строки счета поставщика для времени. Руководители проектов могут использовать строки счетов поставщиков для времени для учета стоимости времени, затраченного субподрядчиками на проекты.

Строки счета поставщика для времени могут ссылаться или не ссылаться на строку субподряда для времени. Если строка счета поставщика для времени ссылается на субподряд, руководители проектов могут сопоставлять и проверять время, за которое выставляется счет, путем сравнения строки счета поставщика с временем, учтенным субподрядчиками и утвержденным руководителями проекта.

В следующей таблице представлена информация о полях в строках счета поставщика для времени.

| Поле | Description | Функциональное воздействие |
| --- | --- | --- |
| Имя. | Название строки счета поставщика для удобства идентификации. | Это название будет отображаться как первый столбец во всех подстановках, основанных на строках счета поставщика. |
| Description | Краткое описание услуг и продуктов, за которые требует оплату поставщик в строке счета поставщика. | None |
| Субподряд | Субподряд, по которому изначально были заказаны услуги. | При выборе субподряда для счета поставщика все строки в счете поставщика будут наследовать этот субподряд. В счете поставщика не может быть строк, котрые ссылаются на разные субподряды. |
| Строка субподряда | Строка субподряда, по которой были заказаны услуги. Список строк субподряда, которые можно выбрать, ограничен строками выбранного субподряда. | Когда в строке счета поставщика для времени выбрана строка субподряда, значения по умолчанию для полей **Проект**, **Роль** и **Резервируемый ресурс** вводятся из соответствующих полей в строке субподряда. Если выбранная строка субподряда имеет значения в полях **Проект**, **Роль** и **Резервируемый ресурс**, значения соответствующих полей в строке счета поставщика не могут отличаться от этих значений. |
| Дата транзакции | Дата, когда фактическое значение затрат по строке счета поставщика будет учтено в проекте. | None |
| Класс проводки | По умолчанию используется значение **Время**. | Значение **Время** указывает, что строка счета поставщика используется для учета суммы счета за время субподрядчика. |
| Project | Название проекта, в котором использовались услуги, за которые выставлен счет. | Это поле является обязательным, и его нельзя оставлять пустым. |
| Задача | Название задачи проекта, в которой использовались услуги, за которые выставлен счет. Это поле доступно, только если выбран проект. Выбор задачи проекта необязателен. | Если это поле оставлено пустым, руководитель проекта может сопоставить строку счета поставщика с временем, учтенным ресурсами субподрядчика в любой задаче проекта. Если строка счета поставщика не ссылается на строку субподряда, а это поле оставлено пустым, фактическое значение затрат, созданное строкой счета поставщика, не будет связано ни с какими фактическими значениями продаж, по которым не выставлены счета. В этом случае, если настроено выставление счетов на основе задач, может получиться так, что счет по затратам не будет выставлен конечному клиенту. |
| Роль | Роль субподрядных ресурсов, за затраченное которыми время выставляется счет. | В этом поле указывается роль, которую выполняют субподрядные ресурсы, затраченное которыми время должно быть оплачено по счету поставщика. |
| Резервируемый ресурс | Имя субподрядчика, за затраченное которым время выставляется счет. Выбирать резервируемый ресурс необязательно. | Если это поле оставлено пустым, руководитель проекта может сопоставить строку счета поставщика с временем, учтенным любым ресурсом, который относится к поставщику, указанному в строке счета поставщика. |
| Количество | Введите количество часов времени, за которые требует оплату поставщик в строке счета. |None |
| Группа единиц измерения | Значение по умолчанию — **Группа единиц измерения времени**, и изменить его нельзя. | None |
| Единица | По умолчанию используется базовая единица часов из группы единиц измерения времени. Вы можете изменить это значение, чтобы покупать время в любой единице из группы единиц измерения времени. | Сочетание значений **Роль** и **Единица** будет использоваться в качестве значения по умолчанию или рассчитанного значения для поля **Цена за единицу** в строке счета поставщика. |
| Цена за единицу | Цена за единицу по умолчанию использует сочетание значений **Роль** и **Единица** из прайс-листа проекта, который применим к дате проводки строки счета поставщика. | Если цена для применимого прайс-листа проекта настроена в единицах, которые отличаются от единиц в строке счета поставщика, система использует преобразование единиц для расчета цены за единицу. |
| Промежуточный итог | Это доступное только для поля чтение рассчитывается как *Количество* &times; *Цена за единицу*, если значения введены и в поле **Количество**, и в поле **Цена за единицу**. Если одно или оба эти поля пусты, вы можете ввести значение в это поле. | None |
| Налог с продаж | Введите сумму налога. | None |
| Общая сумма | Общая сумма по строке счета поставщика, включая налоги. Это поле рассчитывается как *Промежуточный итог* +  *Налог с продаж*. | None |
