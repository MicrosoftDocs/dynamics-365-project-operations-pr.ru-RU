# <a name="vendor-invoice-lines-for-products"></a>Строки счета поставщика для продуктов

[!include [banner](../../includes/dataverse-preview.md)]

_**Относится к:** развертывание Lite — от сделки до счетов-проформ_

Счет поставщика в Microsoft Dynamics 365 Project Operations может содержать строки счета поставщика для продуктов (также называемых материалами). Руководители проектов могут использовать строки счетов поставщиков для продуктов для учета стоимости продуктов, приобретенных в рамках проектов.

Строки счета поставщика для продуктов могут ссылаться или не ссылаться на строку субподряда для материалов. Если строка счета поставщика для продуктов ссылается на субподряд, руководители проектов могут сопоставлять и проверять продукты, за которые выставляется счет, путем сравнения строки счета поставщика с использованием купленных продуктов, учтенным и утвержденным руководителями проекта.

В следующей таблице представлена информация о полях в строках счета поставщика для продуктов.

| Поле | Description | Функциональное воздействие |
| --- | --- | --- |
| Имя. | Название строки счета поставщика для удобства идентификации. | Это название будет отображаться как первый столбец во всех подстановках, основанных на строках счета поставщика. |
| Description | Краткое описание продуктов, за которые требует оплату поставщик в строке счета поставщика. | None |
| Субподряд | Субподряд, по которому изначально были заказаны продукты. | При выборе субподряда для счета поставщика все строки в счете поставщика будут наследовать этот субподряд. В счете поставщика не может быть строк, котрые ссылаются на разные субподряды. |
| Строка субподряда | Строка субподряда, по которой были заказаны продукты. Список строк субподряда, которые можно выбрать, ограничен строками выбранного субподряда. | Когда в строке счета поставщика для продуктов выбрана строка субподряда, значения по умолчанию для полей **Проект**, **Задача** и **Продукт** вводятся из соответствующих полей в строке субподряда. Если выбранная строка субподряда имеет значения в полях **Проект**, **Задача** и **Продукт**, значения соответствующих полей в строке счета поставщика не могут отличаться от этих значений. |
| Дата транзакции | Дата, когда фактическое значение затрат по строке счета поставщика будет учтено в проекте. | None|
| Класс проводки | При выставлении счета за продукты в этом поле должно быть установлено значение **Материал**. | Значение **Материал** указывает, что строка счета поставщика используется для учета суммы счета за закупленные материалы. |
| Project | Название проекта, в котором использовались продукты, за которые выставлен счет. | Это поле является обязательным, и его нельзя оставлять пустым. |
| Задача | Название задачи проекта, в которой использовались продукты, за которые выставлен счет. Это поле доступно, только если выбран проект. Выбор задачи проекта необязателен. | Если это поле оставлено пустым, руководитель проекта может сопоставить строку счета поставщика с купленным продуктом, использованным в любой задаче проекта. Если строка счета поставщика не ссылается на строку субподряда, а это поле оставлено пустым, фактическое значение затрат, созданное строкой счета поставщика, не будет связано ни с какими фактическими значениями продаж, по которым не выставлены счета. В этом случае, если настроено выставление счетов на основе задач, может получиться так, что счет по затратам не будет выставлен конечному клиенту. |
| Выбрать продукт | Укажите, является ли продукт, за который выставляется счет, существующим продуктом из каталога или вписанным продуктом. | Значение по умолчанию вводится из строки субподряда, когда выбрана строка субподряда. |
| Товар | Выберите продукт из каталога. Если продукт представляет собой вписанный продукт, введите название продукта. | Это поле используется для ввода закупочных цен по умолчанию для существующих продуктов. |
| Количество | Введите количество материала, за которое поставщик требует оплату в этой строке счета. | None |
| Группа единиц измерения | Выберите группу единиц измерения единицы, в которой выражено количество, за которое выставляется счет. | None |
| Единица | Значением по умолчанию является базовая единица выбранной группы единиц измерения. Вы можете изменить это значение, чтобы оплачивать продукты в любой единице из выбранной группы единиц измерения. | Сочетание значений **Продукт** и **Единица** будет использоваться в качестве значения по умолчанию или рассчитанного значения для поля **Цена за единицу** в строке счета поставщика. |
| Цена за единицу | Цена за единицу по умолчанию использует сочетание значений **Продукт** и **Единица** из прайс-листа проекта, который применим к дате проводки строки счета поставщика. | None |
| Промежуточный итог | Это доступное только для поля чтение рассчитывается как *Количество* &times; *Цена за единицу*, если значения введены и в поле **Количество**, и в поле **Цена за единицу**. Если одно или оба эти поля пусты, вы можете ввести значение в это поле. | None |
| Налог с продаж | Введите сумму налога. | None |
| Общая сумма | Общая сумма по строке счета поставщика, включая налоги. Это поле рассчитывается как *Промежуточный итог* +  *Налог с продаж*. | None |
