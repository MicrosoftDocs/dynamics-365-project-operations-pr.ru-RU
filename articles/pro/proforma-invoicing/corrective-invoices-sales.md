# <a name="corrective-project-invoices"></a>Корректирующие счета по проекту

_**Относится к:** развертывание Lite — от сделки до счетов-проформ_

Подтвержденный счет по проекту может быть скорректирован для обработки изменений или кредитов по согласованию с заказчиком и менеджером проекта.

Чтобы внести изменения в подтвержденный счет, откройте подтвержденный счет и выберите **Исправить этот счет**. 

> [!NOTE]
> Этот выбор недоступен, если счет по проекту не подтвержден.

Новый черновик счета создается из подтвержденного счета. Все детали строк счета из ранее подтвержденного счета копируются в новый черновик. Ниже приведены некоторые из ключевых моментов, которые нужно знать о деталях строк в новом исправленном счете:

- Все количества обновляются до нуля. Приложение предполагает, что все позиции, по которым выставлен счет, полностью кредитуются. При необходимости вы можете вручную обновить эти количества, чтобы отразить количество, по которому выставляется счет, а не количество, которое кредитуется. На основе введенного количества приложение рассчитывает кредитуемое количество. Эта сумма отражается в фактических данных, которые создаются при подтверждении исправленного счета. Если вы вносите изменения в сумму налога, вы должны ввести правильную сумму налога, а не сумму налога, которая кредитуется.
- Ранее подтвержденные строки контракта на основе продуктов не копируются. Обработка исправлений в счете по проекту на основе продуктов не поддерживается.
- Исправления вех всегда обрабатываются как полные кредиты.
- Суммы предоплаты или авансы могут быть исправлены, если клиенту выставлен счет на неправильную сумму.
- Выверки предоплаты и авансов можно исправить, если неверная сумма была использована для сверки с расходами в ранее подтвержденном счете.

> [!IMPORTANT]
> Детали строк счетов, которые являются исправлениями других уже выставленных счетов, имеют в поле **Исправление** значение **Да**. В счетах, в которых есть исправленные данные строк счетов, есть поле с именем **Есть исправления**, в котором также установлено значение **Да**.
