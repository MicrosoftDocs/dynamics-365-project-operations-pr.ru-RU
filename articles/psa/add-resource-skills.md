<span data-ttu-id="c297d-104">Для проектов ваших клиентов несомненно требуются специалисты с различными навыками, например навыками программирования на языке SQL или навыками в области гибких методологий.</span><span class="sxs-lookup"><span data-stu-id="c297d-104">Your clients’ projects no doubt require a variety of different skills to complete the job, such as SQL programming or agile methodologies.</span></span> <span data-ttu-id="c297d-105">Введите каждый навык, чтобы его можно было в дальнейшем связать с ролями ресурсов и ресурсами.</span><span class="sxs-lookup"><span data-stu-id="c297d-105">Enter each skill so you can associate it with resource roles and resources later on.</span></span>

2. <span data-ttu-id="c297d-107">Нажмите кнопку **Создать**.</span><span class="sxs-lookup"><span data-stu-id="c297d-107">Click **New**.</span></span>  
  
3. <span data-ttu-id="c297d-108">В области **Общие сведения** введите имя навыка в поле "Имя", при необходимости заполните другие поля.</span><span class="sxs-lookup"><span data-stu-id="c297d-108">In the **General** area, enter a name for the skill in Name, and then fill in the other fields as necessary.</span></span>
