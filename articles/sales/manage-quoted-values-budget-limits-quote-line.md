# <a name="project-based-quote-lines-overview"></a>Обзор строк предложения с расценками на основе проекта

_**Относится к:** Project Operations для сценариев на основе ресурсов/без запасов_

Строки предложений с расценками на основе проекта предназначены для помощи в оценке проектной работы по заданию. Структура строки предложения с расценками по проекту расширена для оценок проекта за счет следующих понятий:

- Метод выставления счета
- Сопоставление проекта
- Включенные классы транзакций
- Максимальный лимит
- Настройка возможности оплаты
- Оценка с использованием сведений строки предложения с расценками
- Клиенты строки предложения с расценками

В следующей таблице представлена информация о полях на вкладке **Общее** строки предложения с расценками по проекту. Эти поля помогают настроить основу для подробной, основательной оценки проектной работы.

| **Поле** | **Описание** | **Воздействие на последующие элементы** |
| --- | --- | --- |
| Название | Название строки предложения с расценками, которое должна помочь вам идентифицировать дискретный компонент оцениваемого предложения с расценками. | Копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Метод выставления счета | В предложении с расценками, созданном на основе возможной сделки, это значение копируется из соответствующего поля в строке возможной сделки. Это поле включает две основные модели контрактов, поддерживаемые Dynamics 365 Project Operations:</br>- Фиксированная цена</br>- Время и материал.| Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Project | Используйте это необязательное поле, чтобы определить проект, который будет использоваться для выполнения работы по данному заданию. Когда проект сопоставляется со строкой предложения с расценками, это помогает с настройкой оплачиваемых задач, а также с внесением оценки на основе проекта в строку предложения с расценками в качестве сведений строки предложения с расценками. Если проект не сопоставлен со строкой предложения с расценками по проекту, оценку следует создать вручную, создав сведения каждой строки предложения с расценками. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Включить время | Флаг **Да**/**Нет** указывает, будут ли включены транзакции времени или трудозатраты по выбранному проекту в оценку в этой строке предложения с расценками. Значение **Нет** указывает, что транзакции времени или трудозатраты не будут включены в оценку в этой строке предложения с расценками. Значение **Да** указывает, что транзакции времени или трудозатраты будут включены в оценку в этой строке предложения с расценками. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Включить расход | Флаг **Да**/**Нет** указывает, будут ли включены расходы по выбранному проекту в оценку в этой строке предложения с расценками. Значение **Нет** указывает, что затраты не будут включены в оценку в этой строке предложения с расценками. Значение **Да** указывает, что затраты будут включены в оценку в этой строке предложения с расценками. | Значение этого поля копируется далее в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Включить сбор | Флаг **Да**/**Нет** указывает, будут ли включены сборы по выбранному проекту в оценку в этой строке предложения с расценками. Значение **Нет** указывает, что сборы не будут включены в оценку в этой строке предложения с расценками. Значение **Да** указывает, что сборы будут включены в оценку в этой строке предложения с расценками. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Сумма по предложению | Это сумма, которая будет указана заказчику за все работы, прогнозируемые в этой строке предложения с расценками на основе проекта. В предложении с расценками, созданном на основе возможной сделки, это значение копируется из поля **Бюджет клиента** в строке возможной сделки. Если в строке предложения с расценками по проекту есть сведения о строке, это поле заблокировано для редактирования и суммируется из суммы в сведениях строки предложения с расценками. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Ожидаемый налог | Это редактируемое поле, в котором пользователь может добавить расчетную сумму налога в строку предложения с расценками. Если в строке предложения с расценками по проекту есть сведения о строке, это поле заблокировано для редактирования и суммируется из суммы налога в сведениях строки предложения с расценками. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Сумма по предложению с расценками после налогообложения | Это поле является суммой строки предложения с расценками после уплаты налогов и предназначено только для чтения. Сумма в этом поле рассчитывается как *Сумма по предложению с расценками + Налог*. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Максимальный лимит | Это поле доступно для редактирования и доступно только в строках предложения с расценками на основе проекта, в которых есть способ выставления счетов **Время и материал**. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |
| Бюджет клиента | Это поле доступно для редактирования и копируется из соответствующего поля в строке возможной сделки, если предложение с расценками было создано из возможной сделки. | Значение этого поля копируется в строку контракта по проекту, которая создается из этой строки предложения с расценками, когда предложение с расценками выиграно. |

## <a name="validation-rules-for-fields-on-the-general-tab-of-project-based-quote-lines"></a>Правила проверки для полей на вкладке "Общие" строк предложения с расценками на основе проекта

**Правило 1**: определенный класс транзакций в выбранном проекте может быть включен только в одну строку предложения с расценками на основе проекта.

**Правило 2**: если у возможной сделки есть несколько предложений с расценками, могут быть строки предложения с расценками из разных предложений с расценками, которые все ссылаются на один и тот же проект и включают один и тот же класс транзакций.

**Правило 3**: если предложения с расценками не относятся к одной и той же возможной сделке, они не могут включать один и тот же проект и класс транзакций.
