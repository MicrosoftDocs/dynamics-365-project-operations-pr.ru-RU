# <a name="header-details-for-project-based-contracts"></a>Сведения о заголовке для контрактов на основе проектов

_**Относится к:** Project Operations для сценариев на основе ресурсов/без запасов_

Эта тема предоставляет информацию о полях, которые применяются ко всему контракту проекта, включая настройки, которые влияют на все строки контракта. Также включена информация о контракте, которая обобщена по всем позициям для управления ключевыми показателями эффективности контракта по проекту.

В следующей таблице перечислены поля контракта по проекту, которые уникальны для Dynamics 365 Project Operations или содержат важные изменения в поведении по сравнению с заказами на продажу в Dynamics 365 Sales.

| Поле | Размещение | Описание: | Воздействие на последующие элементы |
| --- | --- | --- | --- |
| Тип | Вкладка **Сводка** (скрытая) | Это поле набора параметров со следующими параметрами:</br>- **На основе работы** (доступно только при установленном Project Operations)</br>- **На основе номенклатуры** (доступно только при установленных Project Operations и Sales)</br>- **На основе сервисного обслуживания** (доступно, если установлен Dynamics 365 Field Service) | В Project Operations значение этого поля по умолчанию равно **На основе работы** и классифицирует контракт как контракт на основе проекта. Контакт должен быть основан на проекте, чтобы включить все расширения и функции, специфичные для проекта. |
| Ответственная компания | Вкладка **Сводка** | Юридическое лицо, которое учитывает затраты и доходы от проектов, связанных с этим контрактом по проекту. Когда контакт создается из предложения с расценками, это поле копируется из соответствующего поля в записи предложения с расценками. | Компания-владелец приравнивается к понятию юридического лица в модуле **Управление проектами и учет** в Project Operations. Все расходы и доходы, связанные с этим проектом, учитываются в главной книге компании-владельца. |
| Клиент | Вкладка **Сводка** | Ссылка на запись компании или учетную запись клиента. Когда контакт создается из предложения с расценками, это поле копируется из соответствующего поля в записи предложения с расценками. | Валюта в контракте по проекту устанавливается по умолчанию в зависимости от валюты клиента. Валюту можно изменить перед сохранением контракта. |
| Менеджер по работе с клиентами | Вкладка **Сводка** | Имя менеджера по обслуживанию клиентов для этой сделки. Когда контакт создается из предложения с расценками, это поле копируется из соответствующего поля в записи предложения с расценками. | Менеджер по работе с клиентами отвечает за управление отношениями с клиентом до завершения проекта. На основе записи резервируемого ресурса, связанного с менеджером по работе с клиентами, контрактная единица задается по умолчанию в контракте по проекту. |
| Единица по контракту | Вкладка **Сводка** | Подразделение, отвечающее за реализацию проектов, связанных с этим контрактом. Когда контакт создается из предложения с расценками, это поле копируется из соответствующего поля в записи предложения с расценками. | Подрядное подразделение — это подразделение компании, которое выполняет проект. У каждой контрактной единицы есть валюта, и эта валюта используется для отчета о предполагаемых и фактических затратах, понесенных в ходе проекта. |
| Прайс-лист продуктов | Вкладка **Сводка** | Этот прайс-лист используется для определения цен по умолчанию в строках контракта на основе продуктов. В раскрывающемся списке параметров для этого поля отображается список прайс-листов, в которых валюта прайс-листа совпадает с валютой в контракте. Когда контакт создается из предложения с расценками, это поле копируется из соответствующего поля в записи предложения с расценками. В контракте по проекту это поле устанавливается по умолчанию из записи учетной записи, но может быть изменено. | У этого поля дальнейшее влияние отсутствует. |
| Валюта | Вкладка **Сводка** | Валюта, в которой указывается стоимость сделки, и валюта, в которой клиенту будет выставлен счет. Когда контакт создается из предложения с расценками, это поле копируется из соответствующего поля в записи предложения с расценками. В контракте по проекту это поле устанавливается по умолчанию из записи учетной записи, но может быть изменено. | После сохранения контракта это поле больше недоступно для редактирования. Это поле используется для задания по умолчанию прайс-листов по продуктам и проектам в контракте. Валюта в контракте используется в соответствии с валютой прайс-листа. |
| Максимальный лимит | Вкладка **Сводка** | Это поле указывает на согласованный верхний предел окончательной стоимости, с которым клиент согласился для этой сделки. | Это ограничение оценивается во время выполнения и применяется ко всем позициям строк и проектам, связанным с этой сделкой. |
| Запрошенная дата доставки | Вкладка **Сводка** | Когда контракт создается из предложения с расценками по проекту, это поле копируется из соответствующего поля предложения с расценками по проекту. | Эта дата используется в качестве конечной даты для создания графиков счетов. |

Следующие ключевые показатели эффективности доступны на вкладке **Производительность по контракту** контракта по проекту.

| Поле | Размещение | Описание: |
| --- | --- | --- |
| Стоимость контракта | Общий контракт | Общая сумма контракта по проекту. |
| Сумма в выставленном счете | Общий контракт | Сумма всех сумм по всем счетам по настоящему контракту. |
| Понесенные затраты | Общий контракт | Сумма всех фактических затрат, зарегистрированных по всем проектам, сопоставленным с контрактом. |
| Валовая прибыль | Общий контракт | Сумма по выставленным счетам - Понесенные затраты на текущую дату/Сумма по выставленным счетам |
| Ожидаемая маржа | Общий контракт | (Стоимость контракта - Предполагаемые затраты) / Стоимость контракта Предполагаемые затраты = сумма всех предполагаемых затрат по всем проектам, сопоставленным с контрактом.|
| Стоимость контракта | Строки на основе проектов | Значение строки контракта. |
| Сумма по выставленным счетам | Строки на основе проектов | Для строки контракта с фиксированной ценой: сумма всех сумм по выставленным счетам для всех фактических результатов продаж по этой строке контракта в различных счетах, созданных для этого контракта. Для строки контракта для времени и материалов: сумма всех сумм по выставленным счетам для всех подлежащих оплате фактических результатов продаж по этой строке контракта в различных счетах, созданных для этого контракта. |
| Понесенные затраты | Строки на основе проектов | Сумма всех фактических затрат, зарегистрированных по всем проектам, сопоставленным со строкой контракта. |
| Валовая прибыль | Строки на основе проектов | (Сумма по выставленным счетам - Понесенные затраты на текущую дату) / Сумма по выставленным счетам |
| Ожидаемая маржа | Строки на основе проектов | (Сумма строки контракта в базовой валюте - Расчетные затраты для строки контракта в базовой валюте) / Сумма строки контракта в базовой валюте |
| Понесенные затраты | Сведения строки на основе проекта | Время: сумма всех сумм фактической стоимости времени, зарегистрированных для этой роли в проекте, сопоставленном со строкой контракта. Расходы: сумма всех сумм фактической стоимости затрат, зарегистрированных для этой категории в проекте, сопоставленном со строкой контракта. |
| Зарегистрированное количество | Сведения строки на основе проекта | Время: все количество фактической стоимости времени для роли в проекте, сопоставленном с этой строкой контракта. Расходы: все количества для категории расходов по фактической стоимости затрат по проекту, сопоставленному с этой строкой контракта. |
| Сумма в выставленном счете | Сведения строки на основе проекта | Для строки контракта с фиксированной ценой это поле остается пустым на уровне сведений и отображается только на уровне строки контракта. Для строки контракта на время и материал расчеты выполняются на уровне сведений. Сведения показывают сумму сумм по всем строкам выручки с выставленными счетами по этой строке контракта, за которые взимается плата. |
| Количество в выставленном счете | Сведения строки на основе проекта | Для строки контракта с фиксированной ценой это поле остается пустым на уровне сведений и отображается только на уровне строки контракта. Для строки контракта на время и материал расчеты выполняются на уровне сведений для времени и расходов. Время: сумма часов по всем строкам выручки с выставленными счетами для этой роли по этой строке контракта, за которые взимается плата. Расходы: все количества для категории расходов по фактической стоимости затрат по проекту, сопоставленному с этой строкой контракта. |
| Стоимость контракта | Строки на основе продуктов | Стоимость строки контракта для этой строки контракта на основе продукта. |
| Сумма в выставленном счете | Строки на основе продуктов | Сумма сумм по всем строкам счетов по этой строке контракта на основе продукта в различных счетах, созданных для этого контракта. |
| Понесенные затраты | Строки на основе продуктов | Сумма всех фактических затрат, зарегистрированных для строки контракта на основе продукта. |
| Валовая прибыль | Строки на основе проектов | Сумма по выставленным счетам - Понесенные затраты на текущую дату/Сумма по выставленным счетам |
| Ожидаемая маржа | Строки на основе продуктов | (Стоимость строки контракта - Оценочные затраты для строки контракта) / Стоимость строки контракта |
