# <a name="estimate-a-projectbased-contract-line"></a>Оценка строки контракта на основе проекта

_**Относится к:** Project Operations для сценариев на основе ресурсов/без запасов_ 

В Dynamics 365 Project Operations строка контракта на основе проекта содержит сведения, которые помогают оценить стоимость и потенциальную выручку от работы, связанной с выполнением строки контракта.

Чтобы оценить строку контракта на основе проекта, перейдите на вкладку **Сведения строки контракта** в пункте **Строка контракта** на основе проекта.  Есть два способа создать смету для строки контракта на основе проекта:

   - Создайте смету прямо в строке контракта, вручную добавив детали строки контракта.
   - Создайте проект и план проекта, затем свяжите проект и задачи со строкой контракта по проекту. Это активирует процесс, с помощью которого вы можете импортировать смету плана проекта в строку контракта на основе компонентов, включенных в строку контракта.

## <a name="create-an-estimate-directly-on-a-projectbased-contract-line"></a>Создание оценки непосредственно из строки контракта на основе проекта

1. Перейдите к строке контракта и выберите вкладку **Сведения строки контракта**. Строки, которые вы создаете на этой вкладке, суммируются и отображаются как **Стоимость контракта** для **Строки контракта**.

## <a name="update-prices-on-contract-line-details"></a>Обновление цен в деталях строки контракта

Если вы измените цены в прайс-листе проекта, который прилагается к контракту, или в прайс-листе себестоимости контрактного подразделения, вы можете обновить цены в деталях отдельной строки контракта, чтобы отразить это изменение. На странице **Контракт** выберите **Пересчитать**. Откроется предупреждение, информирующее вас о том, что цены для всех строк контракта в этом контракте сброшены. Выберите **Да**, чтобы обновить цены для деталей строк контракта для продаж и затрат.

## <a name="access-contract-line-details-for-cost"></a>Доступ к деталям строки контракта для себестоимости

На вкладке **Детали строки контракта** выберите строку в сетке, чтобы отобразить действия на панели инструментов вложенной сетки. Первое действие на панели инструментов вложенной сетки: **Открыть сведения о себестоимости**. Чтобы просмотреть соответствующую ставку затрат и сумму для этих сведений строки контракта, выберите **Открыть сведения о себестоимости**. 

> [!NOTE]
> Изменение ресурсной компании, ресурсной единицы, количества, дат, роли или категорий в деталях строки контракта для пункта **Себестоимость** также изменяет соответствующие значения в деталях строки контракта для пункта **Продажи**.

## <a name="currency-on-contract-line-details-for-cost-and-sales"></a>Валюта в деталях строки контракта для затрат и продаж

Детали строки контракта для пункта **Продажи** устанавливают валюту по умолчанию из прайс-листа проекта, который действует на дату начала деталей строки контракта.

Детали строки контракта для пункта **Себестоимость** устанавливают валюту по умолчанию из прайс-листа контрактного подразделения контракта, который действует на дату начала деталей строки контракта для пункта **Себестоимость**.

Расчеты прибыльности преобразуют суммы для деталей строк контракта для пунктов **Стоимость** и **Продажи** в базовую валюту среды, чтобы указать общую фактическую и расчетную маржу по контракту.

> [!NOTE]
> Ошибки округления валют и изменение маржи могли произойти из-за отсутствия действующих на дату обменных курсов. Используйте эти расчеты в контрактах по проектам только в качестве приблизительных, а не для фактической нормативной или иной отчетности, которая требует более высокой точности округления и учета сроков действия обменных курсов.
