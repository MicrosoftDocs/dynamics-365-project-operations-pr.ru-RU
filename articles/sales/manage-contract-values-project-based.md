# <a name="work-with-projectbased-contract-lines"></a>Работа со строками контракта на основе проекта

Строки контрактов на основе проектов в Dynamics 365 Project Operations предназначены для хранения соглашений об оценке и выставлении счетов для определенных компонентов проектной работы по соглашению. Структура строки контракта на основе проекта расширена для оценок проекта и сценариев выставления счетов за счет следующих концепций:

- Метод выставления счета
- Сопоставление проектов и задач
- Включенные классы транзакций
- Максимальный лимит
- Настройка возможности оплаты
- Оценки с использованием сведений о строке контракта
- Клиенты строки контракта

Следующие поля включены на вкладку **Общие** строк контракта по проекту. Эти поля помогают настроить основу для детальной оценки с самого основания и организации выставления счетов за работу по проекту.

| Поле | Описание: | Воздействие на последующие элементы |
| --- | --- | --- |
| **Название** | Имя строки контракта, которое идентифицирует дискретный компонент контракта, который оценивается. Для контракта по проекту, созданного на основе предложения с расценками, это значение копируется из соответствующего значения строки предложения с расценками по проекту. | Значение этого поля копируется в строку счета по проекту, которая создается из этой строки контракта при создании счета. |
| **Метод выставления счета** | В контракте по проекту, созданному на основе предложения с расценками, это значение копируется из соответствующего поля строки предложения с расценками. Это набор параметров, который представляет две основные модели заключения контрактов, поддерживаемые Project Operations:</br>- **Фиксированная цена**</br>- **Время и материал** | Фактическая транзакция будет обработана в зависимости от метода выставления счетов в указанной строке контракта. Если в строке контракта, на которую ссылается фактическое значение, указан метод выставления счетов по времени и материалам, создаются запись о затратах и фактических продажах без выставления счетов. Если в строке контракта, на которую ссылается фактическое значение, используется метод расчета с фиксированной ценой, создается только фактическая стоимость. |
| **Project** | Используйте это поле, чтобы определить проект, который будет использоваться для выполнения работы по данному заданию. | Значение в этом поле используется вместе с полями **Включенные задачи** и **Включенные классы транзакций** для разрешения ссылки на строку контракта в записи строки фактических данных или оценки. |
| **Включить время** | Флаг указывает, включены ли в эту строку контракта операции по времени или затраты на рабочую силу по выбранному проекту. Значение **Нет** указывает, что операции по времени или затраты на рабочую силу не будут включены в эту строку контракта. Значение **Да** указывает, что они будут. | Это значение поля используется вместе с проектом для разрешения ссылки на строку контракта в записи строки фактических данных или оценки. |
| **Включить расход** | Флаг указывает, будет ли включена стоимость расходов по выбранному проекту в эту строку контракта. Значение **Нет** указывает, что стоимость расходов не будут включена в эту строку контракта. Значение **Да** указывает, что будет. | Это значение поля используется вместе с проектом для разрешения ссылки на строку контракта в записи строки фактических данных или оценки. |
| **Включить сбор** | Флаг указывает, будут ли включены сборы по выбранному проекту в эту строку контракта. Значение **Нет** указывает, что сборы не будут включены в эту строку контракта. Значение **Да** указывает, что они будут. | Это значение поля используется вместе с проектом для разрешения ссылки на строку контракта в записи строки фактических данных или оценки. |
| **Сумма по контракту** | В строке контракта с фиксированной ценой это согласованная сумма, по которой заказчику будет выставлен счет за все компоненты работы, связанные с этой строкой контракта. В строке контракта по времени и материалам эта сумма является оценочным значением, по которому будет выставлен счет клиенту за все компоненты работы, связанные с этой строкой контракта. В контракте по проекту, созданному на основе предложения с расценками, это значение копируется из соответствующего поля строки предложения с расценками. Если в строке контракта на основе проекта есть детали строки, это поле заблокировано для редактирования и суммируется из суммы в деталях строки контракта. | Если в строке контракта есть сведения о строке, это значение можно изменить, изменив суммы в деталях строки. В строке контракта с фиксированной ценой это значение используется для создания суммы до налогообложения для периодических этапов выставления счетов. |
| **Ожидаемый налог** | Пользователь может редактировать это поле, чтобы ввести расчетную сумму налога в строку контракта. Если в строке контракта на основе проекта есть детали строки, это поле заблокировано для редактирования и суммируется из суммы налога в деталях строки контракта. | Если в строке контракта есть сведения о строке, это значение можно изменить, изменив суммы налога в деталях строки. В строке контракта с фиксированной ценой это значение используется для создания налога для периодических этапов выставления счетов. |
| **Сумма по контракту после налогообложения** | Сумма по строке контракта после налогообложения. Это поле доступно только для чтения и рассчитывается как **Сумма контракта + Налог**. | В строке контракта с фиксированной ценой это значение используется для создания периодических этапов выставления счетов. |
| **Максимальный лимит** | Пользователь может редактировать это поле, и оно доступно только для строк контракта на основе проекта, в которых указан метод выставления счетов за время и материалы. | Пользователь может редактировать это поле. Когда фактические затраты времени или расходы ссылаются на эту строку контракта в отношении времени и материалов, сумма фактических расходов оценивается по отношению к максимальному лимиту в этой строке контракта после учета уже потраченных и подтвержденных сумм. |
| **Бюджет клиента** | Это поле доступно для редактирования и копируется из соответствующего поля строки предложения с расценками, если контракт создан из предложения с расценками. | Это поле используется только для информации и не имеет никакого значения для последующей обработки. |
