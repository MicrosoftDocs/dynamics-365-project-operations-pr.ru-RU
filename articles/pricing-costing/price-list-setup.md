# <a name="set-up-price-lists"></a>Настройка прайс-листов

_**Относится к:** Project Operations для сценариев на основе ресурсов/нескладируемых запасов, упрощенное развертывание — от сделки до выставления счетов-фактур_

Прайс-листы в Dynamics 365 Project Operations представляют каталог тарифов. Ставки выражают себестоимость, продажи и ставки выставления счетов. В зависимости от того, указаны ли в прайс-листе ставки себестоимости или продаж или ставки выставления счетов, контекст прайс-листа будет **Продажи** или **Себестоимость**.

Следующие расширения специфичны для Project Operations и применяются к прайс-листам из Dynamics 365 Sales.

- **Контекст**: это поле имеет поддерживаемые значения **Затраты** и **Продажи**. Значение **Покупка** не поддерживается. Установите контекст **Затраты** для создания прайс-листа затрат или установите контекст **Продажи** для прайс-листа продаж. Прайс-листы затрат определяют цену для типа затрат в записях сметы и фактических данных. Прайс-листы продаж определяют цену по оценочным и фактическим записям для типов продаж с невыставленными и выставленными счетами.
- **Единица времени**: это единица времени по умолчанию, для которой цена устанавливается в соответствующей таблице **Цена роли** для данного прайс-листа.
- **Сущность прайс-листа**: это скрытое поле используется Project Operations, чтобы отличать прайс-листы, относящиеся к предложениям с расценками или контрактам, от стандартных и глобально применимых.

## <a name="price-list-header"></a>Заголовок прайс-листа

Следующая таблица включает поля на вкладке **Общие** прайс-листа, которые являются уникальными для Project Operations или имеют существенные изменения в поведении по сравнению с прайс-листами в Sales.

| Поле | Размещение | Описание: | Воздействие на последующие элементы |
| --- | --- | --- | --- |
| Название | Вкладка **Общие** и формы **Быстрое создание** | Идентификатор прайс-листа. | Прайс-лист отображается с этим значением на всех страницах списка и в раскрывающихся списках.|
| Контекст | Вкладка **Общие** и формы **Быстрое создание** | В этом поле можно установить значение **Затраты** или **Продажи**. | Прайс-лист, в котором задано значение **Затраты**, используется для поиска сметы затрат и фактических затрат. Прайс-лист, в котором задано значение **Продажи**, используется для поиска сметы продаж и фактических продаж. Только прайс-листы, для которых задан контекст **Продажи**, могут быть прикреплены к прайс-листам по проекту для клиентов, предложениям с расценками по проектам и контрактам по проектам. |
| Дата начала | Вкладка **Общие** и формы **Быстрое создание** | Дата начала периода, в котором действует прайс-лист. | С полем **Дата окончания** это поле используется для определения того, какой прайс-лист применим для определенной оценки или фактической строки. |
| Дата окончания | Вкладка **Общие** и формы **Быстрое создание** | Дата окончания периода, в котором действует прайс-лист. | С полем **Дата начала** это поле используется для определения того, какой прайс-лист применим для определенной оценки или фактической строки. |
| Валюта | Вкладка **Общие** и формы **Быстрое создание** | Это поле используется для установки валюты по умолчанию для каждой роли, категории или строки позиции прайс-листа, относящейся к этому прайс-листу. | В прайс-листах **Продажи** роли, категории или строки позиций прайс-листа не могут быть созданы ни в какой валюте, кроме этой валюты. В прайс-листах **Затраты** вы можете создать строку цены роли в любой валюте. Валюта, определенная здесь, используется по умолчанию. Настройка пользователя, связанная с ценами ролей, может переопределить это значение, чтобы разрешить настройку ставки оплаты труда в любой валюте. Ставки затрат по категориям и затраты позиций прайс-листа могут быть установлены только в валюте, определенной здесь. |
| Единица времени | Вкладка **Общие** и формы **Быстрое создание** | Это поле используется для установки единицы времени по умолчанию для каждой строки роли, относящейся к этому прайс-листу. | Значение этого поля используется только при настройке цены связанной роли. В прайс-листах **Затраты** и **Продажи** вы можете создать строку цены роли в любой единице времени. Единица времени, определенная здесь, используется по умолчанию. Настройка пользователя, связанная с ценами ролей, может переопределить это значение, чтобы разрешить настройку ставки оплаты труда и ставки выставления счетов в любой единице времени. |
| Описание: | Вкладка **Общие** и формы **Быстрое создание** | Это текстовое поле позволяет вам предоставить многострочное описание прайс-листа. | Это поле отображается в **Связанных** представлениях прайс-листа в различных сущностях, у которых есть связанные прайс-листы. |
